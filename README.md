# Daraz Clone — E-commerce Homepage

A responsive e-commerce homepage inspired by [Daraz.pk](https://www.daraz.pk), built with **HTML**, **CSS**, and **JavaScript**.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (DOM, Events, Smooth Scroll)
- Font Awesome Icons
- Git & GitHub

## Features

- Sticky header with search bar and cart icon
- Image slideshow with auto-play and manual navigation
- App download section
- Flash Sale product row with horizontal scroll
- Categories grid
- "Just For You" product section with Load More
- Login & Sign Up forms
- Footer with Customer Care, payment methods
- Smooth navbar scroll to sections
- Urdu / English language toggle
- Mobile responsive layout

## How to Run Locally

1. Download or clone this repository
2. Open `daraz.html` in your browser
3. No server required (images and assets are local/CDN)
